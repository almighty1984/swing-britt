F1 to switch between Edit and Game

Edit:

    Move mouse to top of window to access menu
    Press del on level in list to delete

    wasd               - move camera / selected
    space              - show tile set
    mouse wheel / 1-5  - brush size
    ctrl               - select    
    x                  - move selected
    c                  - copy selected
    ctrl-z             - undo
    ctrl-shift-z       - redo
    alt                - hide menu
    ctrl-s             - save level
    ctrl-shift-s       - save level as    

Game:

    wasd               - move
    z                  - sprint
    x                  - jump
    c                  - melee
    ctrl               - move camera
